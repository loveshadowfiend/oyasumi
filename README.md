# oyasumi

Manga reading project for Next.js practice that uses MangaDex API

## Live demo

Live on vercel: https://oyasumi-ten.vercel.app/

## Running locally in dev mode

```
git clone https://github.com/loveshadowfiend/oyasumi
cd oyasumi
npm install
npm run dev
```

## Technology stack

-   Next.js
-   TailwindCSS
-   shadcn
-   TanStack Query
-   zustand
